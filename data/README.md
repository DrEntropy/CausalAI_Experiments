# Data Directory

This directory contains datasets used in causal AI experiments.

## Data Organization

- `raw/` - Original, unmodified datasets
- `processed/` - Cleaned and processed datasets
- `synthetic/` - Artificially generated datasets for controlled experiments
- `external/` - Links or references to external datasets

## Common Causal Inference Datasets

Some suggested datasets for causal inference practice:
- Simpson's Paradox examples
- Confounding variable examples
- Instrumental variable datasets
- Randomized controlled trial data

## Data Sources

Consider using datasets from:
- UCI ML Repository
- Causal inference textbooks and papers
- Synthetic data generators for controlled experiments
